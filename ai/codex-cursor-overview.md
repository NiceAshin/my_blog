# AI 编程助手：Codex 与 Cursor 解析

## 1. OpenAI Codex 简介
OpenAI Codex 是一款基于 GPT 架构的代码生成模型，它能够理解自然语言并输出高质量的代码片段。

- **多语言支持**：涵盖 Python, Java, Go, JavaScript 等主流语言。
- **上下文感知**：能够解析现有代码库逻辑，提供智能补全。

## 2. Cursor 编辑器
Cursor 是目前最流行的 AI 原生代码编辑器，深度集成了大模型能力。

- **对话式编程**：直接在编辑器中通过 Chat 提问并生成代码。
- **全库理解**：模型能够索引整个项目，提供全局视角的重构建议。
