# 开源 OCR 工具选型指南

## 1. 评估指标
- **识别准确率**：对多语言及复杂版式的支持。
- **推理性能**：单张图片的处理延迟。
- **部署难度**：是否支持 Docker 及多语言 API。

## 2. 主流工具对比
| 工具 | 核心特点 | 适用场景 |
| --- | --- | --- |
| **PaddleOCR** | 百度出品，中文支持极佳 | 工业级、通用场景 |
| **Tesseract** | 历史悠久，社区资源丰富 | 离线部署、简单文本 |
| **EasyOCR** | Python 实现，安装极简 | 快速原型开发 |
